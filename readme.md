# Cool APP #

----------------------------------------
## RUNNING ##
<b>Starting the environments:</b> <br>

App (frontend):
- cd app
- npm run web

Server (backend):
- cd server
- npm run dev

Prisma:
- cd server
- npx prisma studio

----------------------------------------
## USEFUL COMMANDS ##
<b>Prisma commands:</b> <br>

push changes to the database:
- npx prisma db push

pull changes from the database:
- npx prisma db pull
